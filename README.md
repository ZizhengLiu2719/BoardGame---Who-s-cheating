public url: https://boardgame-who-s-cheating.onrender.com/

## running locally
1. go to /server
2. create file called .env with
```
REDIS_URL='redis://localhost:6379'
```
3. npm i
4. npm run dev
5. open new terminal, start redis: 'redis-server'
6. [localhost](http://localhost:3000/)
